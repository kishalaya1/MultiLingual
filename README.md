# MultiLingual
Multilingual project has been created to showcase implementation of Localization in asp.net core 8 MVC application.This project is designed to support multiple languages. The primary language is English and french,
but additional languages can be added as needed.
The Details of the implementation can be found out in the blog post [here](https://kishalayab.wordpress.com/2025/10/13/multilingual-website-in-asp-net-core-implementing-globalization-and-localization-in-asp-net-core/)
